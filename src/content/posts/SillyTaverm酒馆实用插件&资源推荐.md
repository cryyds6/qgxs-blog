# SillyTaverm实用插件&资源推荐

## 1. 酒馆助手 (Tavern-Helper)
**作者**：@𝚔𝚊𝚔𝚊𝚊🫠、@青空莉正辟谣
**简介**：为 SillyTavern 设计的多功能扩展，支持：
- 在对话中渲染交互式界面（从按钮到复杂游戏界面）
- 用 jQuery 操作 SillyTavern DOM（修改样式、绑定事件等）
- 作为中间层连接 SillyTavern 与外部应用，实现数据/功能联动

**安装文档**：

## 3. 记忆表格（长期聊卡防忘事）
**作者/贡献者**：木悠、蒜鸟、独奏、LSR、阿不思、孤辰辰、Tt
**简介**：增强 SillyTavern 长期记忆，支持自定义角色设定、关键事件、物品等，帮助AI更连贯地推演对话。

**资源链接**：

（麻烦点个星支持~）


## 4. 酒馆助手脚本：字体管理器
**作者**：Angela
> 需先安装「酒馆助手」才能使用

**简介**：快速切换 SillyTavern 字体的工具，按钮位于底部扩展区。

**使用步骤**：
1. 从字体网站复制目标字体的 CSS 代码：https://fonts.zeoseven.com/
2. 打开酒馆助手→脚本库→导入本脚本
3. 将 CSS 粘贴到脚本输入框即可生效

**注意**：Chrome 可能崩溃，建议用 Chrome Beta（更流畅）
**下载地址**：https://wwio.lanzoub.com/i9Zk3373tbcb
